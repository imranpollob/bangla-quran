সূরা ইন্ফিতার

৮২-৬ : হে মানুষ ! কিসে তোমাকে তোমার মহান প্রতিপালক সম্বন্ধে বিভ্রান্ত করিল?

সূরা ইন্ফিতার

৮২-৭ : যিনি তোমাকে সৃষ্টি করিয়াছেন, অতঃপর তোমাকে সুঠাম করিয়াছেন এবং সুসমঞ্জস করিয়াছেন,

সূরা ইন্ফিতার

৮২-৮ : যেই আকৃতিতে চাহিয়াছেন, তিনি তোমাকে গঠন করিয়াছেন।

সূরা ইন্ফিতার

৮২-৯ : না, কখনও না, তোমরা তো শেষ বিচারকে অস্বীকার করিয়া থাক ;

সূরা ইন্ফিতার

৮২-১০ : অবশ্যই আছে তোমাদের জন্য তত্ত্বাবধায়কগণ ;

সূরা ইন্ফিতার

৮২-১১ : সম্মানিত লিপিকরবৃন্দ ;

সূরা ইন্ফিতার

৮২-১২ : তাহারা জানে তোমরা যাহা কর।

সূরা ইন্ফিতার

৮২-১৩ : পুণ্যবানগণ তো থাকিবে পরম স্বাচ্ছন্দ্যে ;

সূরা ইন্ফিতার

৮২-১৪ : এবং পাপাচারীরা তো থাকিবে জাহান্নামে ;

৮২-১৫ : উহারা কর্মফল দিবসে উহাতে প্রবেশ করিবে ;

৮২-১৬ : এবং উহারা উহা হইতে অন্তর্হিত হইতে পারিবে না।

৮২-১৭ : কর্মফল দিবস সম্বন্ধে তুমি কী জান?

৮২-১৮ : আবার বলি, কর্মফল দিবস সম্বন্ধে তুমি কী জান?

সূরা ইন্ফিতার

৮২-১৯ : সেই দিন একের অপরের জন্য কিছু করিবার সামর্থ্য থাকিবে না; এবং সেই দিন সমস্ত কর্তৃত্ব হবে আল্লাহ্‌র।
